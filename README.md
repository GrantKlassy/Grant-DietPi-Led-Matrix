Grant's 64x64 Led Matrix File Dump
==================================================

Overview
--------
A file dump of my MicroSD card for my 64x64 led matrix setup. I wrote some bash scripts to take image files (animated gifs) and convert them to streams for playback on a 64x64 led matrix. 

Bash Scripts
------------
* PLAY_STREAMS.sh
Plays through stream files on loop. This is the main script that I would would run.

* CONVERT_GIFS.sh
I'm pretty sure this converted all the gifs in the cwd to streams

Stream Directories
------------------
* streams
The default streams directory that is used in PLAY_STREAMS.sh

* streams-will
Streams Will made

* streams-100
streams at 100 percent brightness
